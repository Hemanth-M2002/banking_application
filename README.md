# REST API using Spring Boot, Mysql, JPA

This banking application is developed with Spring Boot 3, offering REST APIs for managing banking operations. It uses Spring Data JPA (Hibernate) for database interactions and MySQL for data storage, supporting features like account management, transactions, and customer information.
